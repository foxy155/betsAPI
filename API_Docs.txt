=== Enterprise API Gateway Documentation ===
Version 2.7.0 | RESTful Service Interface

--- AUTHENTICATION REQUIREMENTS ---
* Required for all endpoints except:
  - /documentation
  - /system-status
  - /security.txt

* Authorization Methods:
  1. Query Parameter: ?api_key=<YOUR_KEY>
  2. Header: X-API-KEY: <YOUR_KEY>

* Key Types:
  - ADMIN_MASTER_KEY (Privileged access)
  - DEMO_123 [Disabled] - "Excessive usage"
  - TEMP_456 [Expired] - "Security protocol violation"
